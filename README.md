# XTechon Flight Booking System

A full-stack flight booking application built with:
- React
- Node.js
- Express
- Sequelize
- MySQL

## Features
- User authentication
- Wallet-based booking
- Dynamic flight pricing
- PNR generation
- PDF ticket download with QR code

## How to Run
Backend:
cd backend
npm install
npm start

Frontend:
cd frontend
npm install
npm start
